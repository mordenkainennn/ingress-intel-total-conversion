扇形场规划器

Fanfield Planner 是一个用于 IITC 的扇形场与刺球场规划插件。它允许你选择一个顶点 Portal 和一组底边 Portal，生成双阶段执行计划，并用独立图层查看链接与场的结果。

当前重点： 第一阶段负责构造无交叉的底边图；第二阶段回到顶点并向这张底边图反射，尽量闭合更多场。Summary 还会显示分阶段统计、总 AP，以及第二阶段是否需要提前准备 SBUL。

主要功能

  • 在地图上直接选择一个顶点 Portal 和多个底边 Portal。
  • 生成第一阶段底边搭建路线，并拒绝交叉 link。
  • 根据第一阶段底边图生成第二阶段反射计划。
  • 分别绘制两阶段的链接和场，并输出文字计划。
  • Summary 分别显示 Phase 1、Phase 2 和总计的 link / field、总 AP，以及第二阶段所需 SBUL。

使用流程

  1. 在 toolbox 中打开 Fanfield Planner。
  2. 选择 Select Anchor 后点击顶点 Portal。
  3. 切换到 Select Base Portals 后依次选择底边 Portal。
  4. 点击 Plan Fanfield 生成双阶段计划。
  5. 阅读文字计划，并结合 4 个图层检查结果。

算法概览

第一阶段

底边点先按围绕顶点的角度排序，再选择较短方向进行访问。每到一个底边点，规划器都会尝试回连历史底边点，同时拒绝会造成交叉的 link。

第二阶段

规划器回到顶点后，会根据第一阶段真实构造的底边图决定反射顺序，使每次新增 anchor link 尽量闭合更多场。若第二阶段需要超过 8 条出链，会提前提示所需 SBUL 数量。
