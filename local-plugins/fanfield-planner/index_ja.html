Fanfield Planner

Fanfield Planner は、IITC 上でアンカー型ファンフィールドやピンクッション構造を計画するためのプラグインです。アンカーポータル 1 個と複数のベースポータルを選び、2 フェーズの実行計画と可視化を生成できます。

現在の重点: Phase 1 では交差しないベースグラフを構築し、Phase 2 ではアンカーへ戻ってそのグラフに反射リンクを投げ、できるだけ多くのフィールドを閉じます。Summary には各フェーズの集計、総 AP、SBUL 必要数も表示されます。

主な機能

  • マップ上でアンカーと複数のベースポータルを直接選択できます。
  • Phase 1 の底辺構築では交差するリンクを拒否します。
  • Phase 2 では Phase 1 のベースグラフを使って反射順序を計画します。
  • 各フェーズのリンクとフィールドを別レイヤーに描画します。
  • Summary に Phase 1、Phase 2、合計の link / field、総 AP、SBUL 必要数を表示します。

使い方

  1. toolbox から Fanfield Planner を開きます。
  2. Select Anchor を選んでアンカーポータルをクリックします。
  3. Select Base Portals に切り替えてベースポータルを選びます。
  4. Plan Fanfield を押して 2 フェーズの計画を生成します。
  5. テキスト出力と 4 つのレイヤーを確認し、必要なら再調整します。

アルゴリズム概要

Phase 1

ベースポータルをアンカー周りの角度で並べ、短い方の巡回方向を選びます。各ベースからアンカーへリンクしつつ、交差しない範囲で過去のベースへバックリンクします。

Phase 2

アンカーに戻った後、Phase 1 で構築したベースグラフに対して反射します。必要な出リンク数が 8 を超える場合は、開始前に SBUL 必要数を警告します。

レイヤー

  • Fanfield Plan (Links Phase 1)
  • Fanfield Plan (Fields Phase 1)
  • Fanfield Plan (Links Phase 2)
  • Fanfield Plan (Fields Phase 2)
