Portal DB

Portal DB は、IITCエコシステムの基盤となるインフラストラクチャプラグインです。ブラウザ内にポータルの基本情報を記録するローカルな永続データベースを構築するように設計されています。

マップを閲覧するだけで自動的にデータを収集し、画面外にあるポータルの座標や陣営情報を必要とする他のプラグインにとって、信頼できる「信頼できる唯一の情報源 (Single Source of Truth)」として機能します。

主な機能

  • 永続的なローカルストレージ

    IndexedDB を使用して、ポータルデータ（GUID、緯度、経度、陣営、最終確認日時）を永続的に保存します。このデータは、セッションやブラウザの再起動を超えて保持されます。

  • 受動的なデータ収集

    マップを閲覧するだけで、ポータル情報が自動的にキャプチャされます。生のゲームエンティティデータを直接傍受することで、手動操作を必要とせず、高いパフォーマンスと広範囲なデータ収集を実現します。

  • データ管理

    ツールボックス内の管理インターフェースから、JSONデータベースのエクスポート（バックアップ）やインポート（復元/共有）が可能です。

API リファレンス

このプラグインはグローバルオブジェクト window.plugin.portalDB を介して機能を提供します。すべてのデータアクセス用メソッドは非同期であり、Promise を返します。

getPortal(guid)

指定した GUID のポータルレコードを取得します。

// 例: ポータルが存在するか確認し、座標をログ出力する
window.plugin.portalDB.getPortal('your-portal-guid')
  .then(portal => {
    if (portal) {
      console.log(`発見: ${portal.latE6}, ${portal.lngE6}`);
    } else {
      console.log('ローカルDBにポータルが見つかりません');
    }
  });

getPortalsInBounds(bounds)

指定した地理的境界（Bounds）内のすべてのポータルを取得します。特定のエリア（例：現在のマップ表示範囲）内のポータルを探すのに便利です。

// 例: 現在のマップ表示範囲内の保存済みポータルを検索する
const bounds = map.getBounds(); // Leaflet LatLngBounds オブジェクト
window.plugin.portalDB.getPortalsInBounds(bounds)
  .then(results => {
    console.log(`このエリアで ${results.length} 個のポータルを発見しました。`);
    results.forEach(p => console.log(p.guid, p.team));
  });

戻り値のオブジェクト構造

各メソッドは、以下の標準化された形式のポータルオブジェクトを返します：

{
  "guid": "...",           // String: ポータルの GUID
  "latE6": 35689500,       // Number: 緯度 * 1,000,000
  "lngE6": 139691700,      // Number: 経度 * 1,000,000
  "team": "E",             // String: 'E' (緑), 'R' (青), 'N' (白), 'M' (赤)
  "lastSeen": 1707000000   // Number: 最終更新のタイムスタンプ (ミリ秒)
}

使用方法

  1. Intelマップを閲覧するだけです！プラグインはバックグラウンドで静かに動作し、表示されたすべてのポータルのデータを収集します。
  2. データを管理するには、IITCツールボックスサイドバーの "Portal DB" リンクをクリックしてください。
  3. ダイアログが表示され、ローカルデータベース内のポータルの総数が表示されます。
  4. Export (エクスポート) および Import (インポート) ボタンを使用して、データのバックアップや移行を行ってください。

開発者向けメモ: このプラグインは主にバックエンドサービスとして意図されています。現在描画されていないGUIDについて「このポータルはどこにあるか？」を知る必要があるプラグインを開発している場合、Portal DBがそのソリューションとなります。
