Polygon AP Counter

GitHub でこのプラグインのディレクトリを見る

Polygon AP Counter は、Intel Map 上で作成したポリゴン（多角形）内のポータル、リンク、フィールドを分析するための IITC プラグインです。 両陣営（ENL および RES）がそのエリア内で獲得可能な AP (Action Points) とポータル総数を素早く確認することができます。

クレジット： オリジナルの "Polygon Portal Counter" は 57Cell 氏によって作成されました。 AP 統計機能の追加およびメンテナンスは Cloverjune によって行われています。

主な機能

  • ポータル分析： Draw Tools 描画ツールで作成されたポリゴン内のすべてのポータルを自動的に抽出します。
  • AP 計算： キャプチャ、レゾネーターのデプロイ、および敵リンクやフィールドの破壊によって得られる潜在的な AP を算出します。
  • 陣営別統計： Enlightened と Resistance それぞれの独立した AP 統計を表示します。
  • マップ上の情報表示： ポータル数と各陣営の獲得可能 AP をコンパクトなラベルでマップ上に直接表示します。
  • サマリーテーブル： すべてのポリゴンの詳細な統計リストと合計値をダイアログに表示します。

使用方法

  1. Draw Tools プラグインがインストールされ、有効であることを確認してください。
  2. Draw Tools を使用して、マップ上に 1 つ以上の ポリゴン (Polygon) を描画します。
  3. サイドバーの Toolbox から Polygon AP Counter をクリックします。
  4. 描画された各ポリゴンの統計データを含むダイアログが表示されます。
  5. マップを移動したりポリゴンを編集した場合は、Refresh ボタンを押してデータを更新してください。
  6. テーブル内の View をクリックすると、そのポリゴンがマップの中心になるよう移動します。

AP 計算のロジック

計算ルールは、Available AP Statistics プラグインと同様に、標準的な Ingress の AP 付与ルールに基づいています：

  • キャプチャとデプロイ： 中立ポータルのキャプチャおよび 8 本のレゾネーターのフルデプロイによる AP。
  • 破壊： 敵のレゾネーター、リンク、フィールドの破壊による AP。
  • Machina ポータル： Machina ポータルのリクレイム（再確保）による AP も含まれます。
  • リンクとフィールド： リンクまたはフィールドの頂点の少なくとも 1 つがポリゴン内にある場合にカウントされます。

必要なプラグイン

このプラグインは Draw Tools プラグインを必要とします。ポリゴンが描画されていない場合、データは表示されません。
