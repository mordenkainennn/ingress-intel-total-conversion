多边形 AP 统计 (Polygon AP Counter)

在 GitHub 上查看此插件目录

Polygon AP Counter 是一款 IITC 插件，专门用于分析 Intel 地图上封闭多边形区域内的 Portal、链接 (Link) 和控制场 (Field)。 它可以快速预览两个阵营（启蒙军 ENL 和 抵抗军 RES）在该区域内可获得的潜在 AP (Action Points)，并统计 Portal 的总数。

鸣谢： 原始多边形 Portal 统计插件由 57Cell 编写。 由 Cloverjune 添加了 AP 统计功能并进行维护。

核心功能

  • Portal 分析： 自动识别所有位于 Draw Tools 插件绘制的多边形内的 Portal。
  • AP 计算： 计算包括占领、部署共振器以及摧毁敌方链接和控制场在内的潜在 AP 收益。
  • 阵营统计： 分别查看 启蒙军 和 抵抗军 的独立 AP 数据。
  • 地图交互气泡： 在地图上直接以紧凑的标签显示 Portal 数量和阵营 AP。
  • 摘要表格： 提供包含每个多边形统计数据的详细列表及总计摘要。

使用方法

  1. 确保已安装并启用 Draw Tools 插件。
  2. 使用 Draw Tools 在地图上绘制一个或多个 多边形 (Polygon)。
  3. 在左侧工具栏 (Toolbox) 中点击 Polygon AP Counter。
  4. 弹出对话框将显示每个已绘制多边形的统计数据。
  5. 如果移动了地图或修改了多边形，点击 Refresh 刷新数据。
  6. 点击表格中的 View 可以将地图中心定位到对应的多边形。

AP 计算逻辑

计算规则遵循标准的 Ingress AP 规则，类似于 Available AP Statistics 插件：

  • 占领与部署： 包含占领中立 Portal 并完全部署 8 个共振器的 AP。
  • 摧毁： 包含摧毁敌方共振器、链接和控制场的 AP。
  • Machina Portal： 包含收复 Machina 控制的 Portal 的 AP。
  • 链接与控制场： 如果链接或控制场的至少一个顶点在多边形内，则计入统计。

运行要求

此插件依赖于 Draw Tools 插件。如果没有绘制任何多边形，插件点击后将不显示任何有效数据。
