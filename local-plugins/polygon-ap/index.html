Polygon AP Counter

View this plugin directory on GitHub

Polygon AP Counter is an IITC plugin designed to analyze portals, links, and fields within closed polygons drawn on the map. It provides a quick overview of the potential Action Points (AP) available for both factions (ENL and RES) and the total portal count.

Credits: Original "Polygon Portal Counter" by 57Cell. Enhanced with AP statistics and maintained by Cloverjune.

Core Features

  • Portal Analysis: Automatically identifies all portals inside polygons drawn with the Draw Tools plugin.
  • AP Calculation: Computes potential AP gains considering portal capture, resonator deployment, and destruction of enemy links and fields.
  • Faction-Specific Stats: View independent AP statistics for both Enlightened and Resistance.
  • Interactive Map Bubbles: Shows portal count and faction AP in compact labels directly on the map.
  • Summary Table: Provides a detailed list of all polygons with their respective stats and a total summary.

How to Use

  1. Ensure the Draw Tools plugin is installed and active.
  2. Draw one or more polygons on the map using Draw Tools.
  3. Click Polygon AP Counter in the toolbox sidebar.
  4. A dialog will appear with the statistics for each drawn polygon.
  5. Click Refresh to update data if you move the map or edit the polygons.
  6. Click View in the table to center the map on a specific polygon.

AP Calculation Logic

The calculation follows the standard Ingress AP rules, similar to the Available AP Statistics plugin:

  • Capture & Deploy: AP for capturing a neutral portal and fully deploying 8 resonators.
  • Destruction: AP for destroying enemy resonators, links, and fields.
  • Machina Portals: Includes reclaim AP for Machina-controlled portals.
  • Links & Fields: Links and fields are counted if at least one of their vertices is within the polygon boundary.

Requirements

This plugin requires the Draw Tools plugin to be installed. Without any drawn polygons, the plugin will not show any data.
