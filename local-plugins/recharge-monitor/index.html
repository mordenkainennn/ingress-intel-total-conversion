Recharge Monitor & Decay Predictor

This plugin is an essential utility for agents who manage multiple portals across different regions. It allows you to monitor portal health in real-time and provides intelligent predictions for when a portal will fully decay, even when it's out of your current map view.

By integrating with COMM data and other logging plugins, it ensures you never miss a recharge deadline again.

Features

  • Energy Monitoring Watchlist

    Add specific portals to your personal watchlist via the portal details pane. Keep track of their current health, last seen status, and deployment timestamps in a centralized dashboard.

  • Intelligent Decay Prediction

    For portals that are not currently visible on your map, the plugin calculates an Estimated Decay date based on the standard 15% daily energy loss. It provides a reliable "countdown" to when the portal will be neutralized.

  • Multi-Source History Sync

    Recover accurate deployment times automatically using two powerful methods:

    • COMM Backfilling: Scans IITC's internal chat cache for capture and deployment events.
    • Activity Log Integration: Syncs historical data from the Player Activity Log plugin using precise GUID and coordinate matching.

  • Visual Recharge Alerts

    The "Recharge Mon" button in the IITC toolbox dynamically changes color and displays a warning count (e.g., ‚ö†Ô∏è Recharge (3)) when any watched portal drops below 30% health.

  • Localization & Ease of Use

    Displays all timestamps in your local time zone. Includes a user-friendly interface for manually correcting deployment times if needed.
