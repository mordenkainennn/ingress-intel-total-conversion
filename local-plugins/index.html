IITC Plugin Directory

A collection of custom-built and enhanced plugins for IITC on Niantic's Ingress Intel Map.

This directory highlights tools for Ingress agents who need portal tracking, field planning, drone route planning, activity analysis, recharge monitoring, and other map-focused IITC utilities.

Browse the GitHub repository

Plugin Directory

Each page below explains what the plugin does, who it is useful for, and how it supports planning or analysis on the Ingress Intel Map.

Portal DB

Builds a persistent local portal database for IITC, with move detection and update statistics for Ingress Intel analysis.

Portal Afterimage

Draws a memory layer of portals you have already seen when official Ingress Intel markers are hidden by zoom.

Player Activity Log

Records agent actions from COMM and visualizes movement trails for Ingress activity review on the map.

Recharge Monitor

Monitors portal health in real time and predicts decay windows for defense planning on the Intel Map.

Uniques Tools

Tracks visits, captures, and drone movements with official history import for long-term Ingress progress tracking.

Portal Coverage Layers

Uses Uniques Tools history to show separate Unvisited and Uncaptured portal layers for direct cleanup and route planning.

Homogeneous Fields

Helps plan and visualize homogeneous control fields for structured Ingress fielding operations.

Drone Flight Planner

Plans drone routes with S2 mechanics, key-jump constraints, and JSON import or export for Intel workflows.

Fanfield Planner

Plans two-phase fanfields with non-crossing base links, anchor reflections, and clear phase-by-phase summaries.

All Portal Names

Forces portal name display at chosen zoom levels so Ingress Intel labels stay visible even with overlap.

Polygon AP Counter

Analyzes user-drawn polygons to estimate portal counts, links, fields, and potential ENL or RES AP inside target areas.

Why Use These IITC Plugins

These tools are designed for agents who use IITC to extend the Ingress Intel Map beyond the default experience. They focus on portal memory, operational planning, field construction, COMM-based review, and quality-of-life improvements for daily play.

  • Portal tracking and local data storage for better map awareness.
  • Planning support for drone jumps, fanfields, and homogeneous fields.
  • Agent utilities that turn Intel Map activity into more actionable information.

FAQ

  • What is IITC? IITC is an enhanced interface for the Ingress Intel Map used by agents who want more control, visibility, and planning tools.
  • Are these plugins for Niantic's Ingress? Yes. This directory is specifically about plugins for IITC on Niantic's Ingress Intel Map, not Kubernetes or networking software.
  • Where can I review source files and plugin code? Use the GitHub repository link above to inspect the source, plugin scripts, and update history.
Note: These plugins are based on locally received data only and do not violate Niantic's TOS.
