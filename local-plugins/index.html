IITC Plugin Directory

A collection of custom-built and enhanced plugins for Ingress agents.

Portal DB

The core infrastructure. Persistent database with move detection and update statistics.

Portal Afterimage

Draws a "memory layer" of portals you've seen when official markers are hidden by zoom.

Player Activity Log

Logs agent activities from COMM and visualizes movement trails on the map.

Recharge Monitor

Real-time portal health monitoring and intelligent decay prediction.

Uniques Tools

Enhanced tracking for visits, captures, and drone movements with official history import.

Homogeneous Fields

Advanced tool for planning and visualizing homogeneous control fields.

Drone Flight Planner

Plans drone routes with S2 mechanics, key-jump constraints, and JSON import/export.

All Portal Names

Force display of all portal names regardless of overlap at user-defined zoom levels.
Note: These plugins are based on locally received data only and do not violate Niantic's TOS.
