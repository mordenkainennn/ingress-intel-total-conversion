Homogeneous Fields

Niantic Ingress Intel Map 向けの高度な IITC フィールド計画プラグインです。

Homogeneous Fields は、もともと 57Cell によって作られたフィールド計画ツールで、選択したアンカーポータルから複雑なフィールド作戦を自動生成できます。均一多重フィールド、最大フィールド、Cobweb 形式の計画に対応し、地図上のオーバーレイと実行用のテキスト手順を同時に提供します。

このプラグインが向いている人

リンクを一本ずつ手作業で組むのではなく、大規模なフィールド作戦を体系的に設計したいエージェント向けです。多層フィールド、三角形内ポータルの選別、実行順、鍵数、移動距離まで含めて考えたい場面に向いています。

主な機能

  • 複数の計画モード: Homogeneous Fields (HCF)、General Maximum Fielding、Cobweb Plan を選べます。
  • 対話型の計画ダイアログ: コーナーポータル選択、アンカー画像プレビュー、ロック、設定調整を一つの画面で行えます。
  • ポータル選別: Included / Excluded リストで、計画に使うポータルを細かく調整できます。
  • 地図上の可視化: 計画したリンクとフィールドを Intel Map 上に描画し、色も調整できます。
  • 詳細な実行出力: リンク順、訪問順、SoftBank 数、キー数、総移動距離に加え、この計画で得られる総 AP の見積もりも出力します。
  • 他プラグイン連携: 生成したリンク計画を DrawTools や Arc に出力できます。

計画モード

  • Homogeneous Fields (HCF): 指定層数の均一多重フィールドを構築します。
  • General Maximum Fielding: 利用可能なポータルから最大数のフィールドを目指します。
  • Cobweb Plan: 蜘蛛の巣型の構造を作り、密度重視の展開に向いています。

使い方

  1. IITC ツールボックスで Plan Fields を押します。
  2. 地図上で 3 つのポータルを主要アンカーまたは三角形の角として選びます。
  3. 計画ダイアログを開き、必要に応じてコーナーを固定します。
  4. フィールドタイプ、層数、ジオメトリモードを選びます。
  5. 必要ならポータルを Included と Excluded の間で移動します。
  6. Find Fielding Plan を押して計画を生成します。

出力内容

  • 最適なポータル訪問順。
  • 各ポータルから投げるリンクの番号付きリスト。
  • 主要アンカーに必要な SoftBank 数。
  • ポータルごとの必要キー数。
  • 総移動距離、総 AP、全体統計。

由来と保守: これは 57Cell のオリジナル Field Planner を基に改良されたもので、後に Heistergand や cloverjune らによって拡張・維持されています。

FAQ

  • このプラグインは均一フィールド専用ですか？ いいえ。均一多重フィールドだけでなく、最大フィールドや Cobweb 計画にも対応しています。
  • ポータルを除外できますか？ はい。主三角形を決めた後、Included から Excluded へ移動できます。
  • 見た目だけの計画ですか？ いいえ。リンク順、キー数、SoftBank 数、移動距離に加え、総 AP 見積もりを含む詳細なテキスト手順も出力します。
