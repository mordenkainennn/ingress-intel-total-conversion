Homogeneous Fields

An advanced field planning plugin for IITC on Niantic's Ingress Intel Map.

Homogeneous Fields, originally created by 57Cell, helps Ingress agents design layered field plans from selected anchor portals. It supports classic homogeneous control fields, general maximum fielding, and cobweb-style operations, with both visual overlays and step-by-step textual output.

Who This Plugin Is For

This plugin is for agents who plan structured fielding operations rather than drawing links manually one by one. It is especially useful for layered field projects, portal filtering inside a main triangle, and operation plans that need clear link order, key counts, and travel estimates.

Key Features

  • Multiple planning modes: Choose between Homogeneous Fields (HCF), General Maximum Fielding, and Cobweb Plan depending on the operation style.
  • Interactive planning dialog: Select corner portals, preview anchor images, lock selections, and adjust plan settings from one workspace.
  • Portal inclusion control: Move portals between included and excluded lists to refine the final plan.
  • Visual overlays: Draw the planned links and fields directly on the Intel Map and adjust overlay color as needed.
  • Detailed execution output: Get link order, visitation order, SoftBank counts, key counts, and total travel distance.
  • Plugin integration: Export the generated link plan to DrawTools and Arc.

Planning Modes

  • Homogeneous Fields (HCF): Builds a layered field structure with a chosen number of layers.
  • General Maximum Fielding: Tries to maximize the total number of fields from the available portals.
  • Cobweb Plan: Generates a spiderweb-style structure that is useful for area denial or visual density.

How to Use Homogeneous Fields

  1. Click Plan Fields in the IITC toolbox.
  2. Select three portals on the map as the main anchors or triangle corners.
  3. Open the planning dialog and lock the chosen corners if needed.
  4. Select the desired field type, layer count, and geometry mode.
  5. Optionally move portals between the included and excluded lists.
  6. Click Find Fielding Plan to generate the final result.

What the Output Includes

  • The optimal order for visiting portals during the operation.
  • A numbered list of links to throw from each portal.
  • SoftBank counts for major anchors.
  • Key counts per portal.
  • Total travel distance and overall plan statistics.

Origin and maintenance: This is an improved version of the original field planner by 57Cell, later extended by contributors such as Heistergand and cloverjune inside this IITC plugin collection.

FAQ

  • Is this plugin only for homogeneous fields? No. Homogeneous fields are one major mode, but the plugin also supports general maximum fielding and cobweb plans.
  • Can I exclude portals from the plan? Yes. After choosing the main triangle, you can move portals from the included list to the excluded list.
  • Does the plugin only show a visual plan? No. It also produces a detailed text-based plan with link order, keys, SoftBanks, and travel information.
