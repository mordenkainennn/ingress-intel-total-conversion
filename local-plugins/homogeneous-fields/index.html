57Cell's Field Planner

This plugin is an advanced tool for designing and executing complex fielding operations. Originally created by 57Cell, it has been significantly enhanced by contributors like Heistergand and mordenkainennn to become a powerful asset for strategic agents.

It allows for the automatic calculation of linking and fielding plans based on selected anchor portals, saving hours of manual planning.

Features

  • Multiple Fielding Strategies

    Choose between three distinct fielding algorithms to match your strategic goals:

    • Homogeneous Fields (HCF): Creates a perfectly layered field with a specified number of layers.
    • General Maximum Fielding: Aims to create the maximum number of fields from a given set of portals.
    • Cobweb Plan: Generates a spiderweb-like field pattern, excellent for area denial.

  • Interactive Planning UI

    A comprehensive dialog, launched from the toolbox, serves as the command center. From here you can select corner portals, see their image previews, lock them in, and fine-tune your plan.

  • Portal Management

    After selecting three corner portals, the plugin automatically populates a list of all portals inside the main triangle. You can easily move portals between an "Included" and "Excluded" list to control which ones are used in the final plan.

  • Visual Feedback & Customization

    The generated plan is drawn directly onto the map, showing all required links and fields. You can change the color of the plan's overlay in real-time. Selected corner portals get animated highlights for easy identification.

  • Detailed Step-by-Step Output

    The plugin generates a detailed text-based plan that can be copied and shared. The plan includes:

    • The optimal portal visitation order.
    • A numbered list of links to throw from each portal.
    • The number of SoftBanks required for each anchor.
    • A summary of keys needed for each portal.
    • Total travel distance between portals.

  • Integration with Other Plugins

    For even greater efficiency, you can export the generated link plan directly to the DrawTools and Arc plugins with a single click.

How to Use

  1. Click the "Plan Fields" button in the IITC toolbox to get started.
  2. Select three portals on the map to serve as your main anchors/corners.
  3. The planning dialog will open. Here you can:
    • Lock your selected corners.
    • Choose your desired Field type (HCF, General, Cobweb).
    • Set the number of Layers for an HCF plan.
    • Choose the plan Geometry (Random or Perfect).
  4. (Optional) In the right-hand panel, move any portals you don't want to use from the "Included" list to the "Excluded" list.
  5. Click the "Find Fielding Plan" button.
  6. The plugin will calculate the optimal plan and display the visual overlay on the map and the text steps in the dialog.

Origin: This plugin is an improved version based on the original script by 57Cell. It is now maintained and enhanced by mordenkainennn within this IITC-CE plugin collection.
