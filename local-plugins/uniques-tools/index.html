Uniques Tools

An enhanced portal history plugin for IITC on Niantic's Ingress Intel Map.

Uniques Tools replaces the stock IITC-CE Uniques plugin with richer visited, captured, scanned, and drone tracking. It helps Ingress agents manage portal history, import official records in view, and work with clearer map highlights and drone location tools.

View this plugin directory on GitHub | See all IITC plugins

Who This Plugin Is For

Uniques Tools is built for agents who want more control over portal history than the stock IITC highlighter provides. It is especially useful if you track drone visits, scout controller progress, or local visited and captured status while planning routes and cleanup work.

Key Features

  • Stock Uniques replacement: Expands the standard visited and captured workflow with a standalone toolbox, richer controls, and clearer state management.
  • Drone support: Track the last three drone-visited portals, jump to the latest location, and show both persistent and temporary 550m drone range circles on the Intel Map.
  • Scout tracking: Adds a manual Scanned state so you can track scout controller progress alongside visited and captured status.
  • Official history import: Import visited, captured, and scanned history from official Ingress data for portals currently visible in your map view.
  • Portal counts summary: Adds a toolbox button that shows your current totals for Visited, Captured, Scanned, and Drone Visit.
  • Improved highlighting: Uses clearer visual states for unvisited, scanned-only, drone-only, visited, and captured portals.

How It Works

When you open a portal, the details panel shows four checkboxes: Visited, Captured, Scanned, and Drone. These values are stored locally and can also sync if you use the IITC Sync plugin.

The Uniques Tools toolbox dialog adds quick actions for importing official history, showing status counts, clearing the saved drone location, and jumping back to the last drone-visited portal. The action buttons are arranged in a cleaner two-column layout, making the dialog easier to use during regular play and post-operation review.

Highlight States

  • Red: Unvisited portal.
  • Yellow: Scout controlled but not visited or captured.
  • Cyan: Drone visited but not visited or captured in person.
  • Purple: Visited but not captured.
  • No highlight: Captured portal.

How to Use Uniques Tools

  1. Select a portal and update Visited, Captured, Scanned, or Drone directly from the details panel.
  2. Open the Uniques Tools button in the IITC toolbox for import and drone location actions.
  3. Use Import History while viewing an area to align local data with official portal history already available to your client.
  4. Use Find Last Location when you want to return to the most recently marked drone portal.

Compatibility note: This plugin is designed to replace the stock IITC-CE Uniques plugin. Disable the built-in highlighter to avoid duplicate UI elements and conflicting behavior.

Portal DB recommended: Installing Portal DB improves the reliability of Find Last Location and makes off-screen portal lookups smoother.

FAQ

  • Do I need to disable the stock Uniques plugin? Yes. Running both at the same time can create duplicate controls and confusing highlight behavior.
  • Does the plugin support drone tracking? Yes. It stores the last three drone-marked portals, draws location markers, and provides range guidance with 550m circles.
  • Can I import official game history? Yes. The plugin can import visited, captured, and scanned status for portals currently visible in the Intel Map view.
