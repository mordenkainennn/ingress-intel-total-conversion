Uniques Tools

Uniques Tools is an enhanced version of the standard "Uniques" plugin built into IITC-CE. It expands upon the original's functionality to provide agents with more powerful and detailed tracking of their portal history.

The original plugin's purpose is to help agents track which portals they have visited and captured. This plugin takes that concept to the next level.

New Features

  • Full Drone Support

    Track portals visited via Drone View. The plugin automatically remembers the last 3 drone locations, displays special markers on the map (last, previous, oldest), and allows you to quickly jump to the last location using the "Find Last Location" button. Additionally, a 550m drone range circle is drawn around the currently selected portal, providing a visual guide for maximum flight distance.

  • Scout Controller Tracking

    A new "Scanned" status to manually track portals you have scout controlled. This helps in planning and tracking for the Scout Controller medal.

  • Import Official History

    A powerful feature to import your visited, captured, and scanned history directly from Ingress's official data for all portals currently in view. Access it via the "Uniques Tools" dialog.

  • Enhanced Highlighting

    A more detailed and clear color scheme for the map highlighter:

    • Red: Unvisited
    • Yellow: Scout Controlled (but not visited/captured)
    • Cyan: Drone Visited (but not visited/captured)
    • Purple: Visited (but not captured)
    • No Highlight: Captured

  • Centralized Tools Dialog

    A convenient "Uniques Tools" button in the IITC toolbox provides access to all major functions like importing history and managing drone data.

How to Use

Portal Details Panel

When you select a portal, you will see four checkboxes below the portal image: Visited, Captured, Scanned, and Drone. Simply check the appropriate box to manually update its status. The data is saved locally and can be synced if you use the "Sync" plugin.

"Uniques Tools" Dialog

Find the "Uniques Tools" button in the IITC toolbox on the right. Clicking it opens a dialog where you can:

  • Import History: Click this to update your local data with official game history for portals in the current view.
  • Clear Drone History: Removes all drone location markers and history data.
  • Find Last Location: Pans and zooms the map to the most recent portal visited by your drone.

Important Note: This plugin is a replacement for the stock "Uniques" plugin. To avoid conflicts and duplicate UI elements, it is highly recommended to disable the built-in "Uniques" plugin from the IITC Settings panel (Layers -> Highlighter -> Uniques). Our plugin will display a warning and turn its toolbox button red if it detects the stock plugin is also active.
