Uniques Tools

适用于 Niantic Ingress Intel Map 的 IITC Portal 历史增强插件。

Uniques Tools 用来替代 IITC-CE 自带的 Uniques 插件，提供更完整的 visited、captured、scanned 与 drone 状态管理。它适合需要整理 Portal 历史、导入官方记录、追踪无人机位置和查看更清晰地图高亮的 Ingress 特工。

在 GitHub 上查看此插件目录 | 查看全部 IITC 插件

适合哪些用户

如果你觉得原版 Uniques 只能满足最基础的 visited 和 captured 统计，那么这个插件会更适合你。它尤其适合需要追踪无人机访问、侦察官进度、手动补录 Portal 状态，或者需要在作业后回顾本地 Portal 历史的玩家。

核心功能

  • 替代原版 Uniques： 使用独立工具箱和更完整的状态控制，扩展原版 visited / captured 工作流。
  • 完整无人机支持： 记录最近三个 drone 访问过的 Portal，支持快速跳转到最后位置，并显示常驻和临时的 550 米 无人机航程圈。
  • Scout 状态追踪： 增加手动 Scanned 状态，方便追踪 Scout Controller 相关进度。
  • 官方历史导入： 可将当前地图视野内 Portal 的官方 visited、captured 和 scanned 历史导入到本地记录中。
  • 状态计数统计： 在工具箱中新增按钮，可快速查看当前 Visited、Captured、Scanned 和 Drone Visit 的 Portal 总数。
  • 更清晰的高亮规则： 能更直观地区分未访问、仅扫描、仅无人机访问、已访问未占领和已占领等状态。

工作方式

选择一个 Portal 后，详情面板会显示 Visited、Captured、Scanned 和 Drone 四个复选框。你可以直接在这里手动更新状态，数据会保存在本地；如果启用了 Sync 插件，也可以进一步同步。

Uniques Tools 工具箱对话框还提供导入官方历史、查看状态计数、清除无人机最后位置、返回最后一个 drone Portal 等操作。现在按钮区域也整理成了每行两个按钮的布局，因此无论是日常使用还是作战后回顾都会更顺手。

地图高亮说明

  • 红色： 未访问 Portal。
  • 黄色： 已扫描，但未访问或占领。
  • 青色： 仅通过无人机访问，但未实地访问或占领。
  • 紫色： 已访问，但未占领。
  • 无高亮： 已占领 Portal。

如何使用

  1. 选择 Portal 后，在详情面板中更新 Visited、Captured、Scanned 或 Drone 状态。
  2. 打开 IITC 工具箱中的 Uniques Tools 按钮，访问导入与无人机位置相关功能。
  3. 在浏览某个区域时使用 Import History，把当前视野里官方可见的历史记录同步进本地数据。
  4. 需要回到最新无人机位置时，使用 Find Last Location 快速跳转。

兼容性提示： 这个插件的设计目标就是替代 IITC-CE 自带的 Uniques。为了避免重复按钮、重复高亮和界面冲突，请关闭原版 Uniques 插件。

推荐搭配： 安装 Portal DB 后，Find Last Location 的离屏 Portal 定位会更稳定，整体体验也更顺畅。

常见问题

  • 必须关闭原版 Uniques 吗？ 建议关闭。否则很容易出现重复控件和不一致的高亮行为。
  • 支持无人机访问记录吗？ 支持。插件会保存最近三个 drone 位置，并显示位置标记和 550 米航程圈。
  • 可以导入官方历史吗？ 可以。对于当前 Intel Map 视野中可见的 Portal，插件可以导入其官方 visited、captured 和 scanned 历史。

相关插件

  • Player Activity Log 侧重特工行为历史，能和 Uniques Tools 的 Portal 历史记录形成互补。
