Uniques Tools

Uniques Tools は、IITC-CEに組み込まれている標準プラグイン「Uniques」の強化版です。オリジナルの機能を拡張し、エージェントにポータル履歴のより強力で詳細な追跡機能を提供します。

オリジナルプラグインの目的は、エージェントがどのポータルを訪問したか (visited)、キャプチャしたか (captured) を追跡するのを助けることです。このプラグインはその概念を次のレベルに引き上げます。

新機能

  • 完全なドローン (Drone) サポート

    ドローンビューを介して訪問したポータルを追跡します。プラグインは最後の3つのドローン位置を自動的に記憶し、マップ上に特別なマーカー（最後、前回、最古）を表示し、「最後の位置を検索」ボタンを使用して最後の位置にすばやくジャンプできます。さらに、現在選択されているポータルの周りに550mのドローン航続範囲円が描画され、最大飛行距離の視覚的なガイドを提供します。

  • スカウトコントローラー (Scout Controller) 追跡

    スカウトコントロールしたポータルを手動で追跡するための新しい「スキャン済み」 (Scanned) ステータス。これは、スカウトコントローラーメダルの計画と追跡に役立ちます。

  • 公式履歴のインポート

    現在表示されているすべてのポータルについて、Ingressの公式データから訪問、キャプチャ、スキャンの履歴を直接インポートする強力な機能。「Uniques Tools」ダイアログからアクセスできます。

  • 強化されたハイライト表示

    マップハイライターのための、より詳細で明確なカラースキーム：

    • 赤: 未訪問
    • 黄: スキャン済み (訪問/キャプチャはしていない)
    • シアン: ドローンで訪問 (訪問/キャプチャはしていない)
    • 紫: 訪問済み (キャプチャはしていない)
    • ハイライトなし: キャプチャ済み

  • 統合されたツールダイアログ

    IITCツールボックス内の便利な「Uniques Tools」ボタンから、履歴のインポートやドローンデータの管理など、すべての主要機能にアクセスできます。

使用方法

ポータル詳細パネル

ポータルを選択すると、ポータル画像の下に4つのチェックボックスが表示されます：Visited (訪問)、Captured (キャプチャ)、Scanned (スキャン)、Drone (ドローン)。適切なボックスにチェックを入れるだけで、そのステータスを手動で更新できます。データはローカルに保存され、「Sync」プラグインを使用している場合は同期することもできます。

「Uniques Tools」ダイアログ

右側のIITCツールボックスで「Uniques Tools」ボタンを見つけます。それをクリックするとダイアログが開き、以下の操作ができます：

  • Import History (履歴のインポート): これをクリックして、現在のビュー内のポータルのローカルデータを公式ゲーム履歴で更新します。
  • Clear Drone History (ドローン履歴のクリア): すべてのドローン位置マーカーと履歴データを削除します。
  • Find Last Location (最後の位置を検索): マップをパンおよびズームして、ドローンが最後に訪れたポータルに移動します。

重要: このプラグインは、標準の「Uniques」プラグインの代替品です。競合やUI要素の重複を避けるため、IITCの設定パネル (Layers -> Highlighter -> Uniques) で標準の「Uniques」プラグインを無効にすることを強く推奨します。このプラグインは、標準プラグインがアクティブであることを検出すると、警告を表示し、ツールボックスのボタンを赤くします。

変更履歴

バージョン 1.6.1

  • 更新: プラグインを内部的に「Uniques Tools」に名称変更しました（旧称「uniques-drone-final」）。
  • 更新: 内部ID、名前空間、ローカルストレージキーを新しい名前に合わせて更新しました。
  • 修正: 既存のユーザーデータは移行されません。プラグインは新しい空のデータで開始します。

バージョン 1.6.0

  • 新規: 選択したポータルの周りに550mのドローン航続範囲円を追加し、最大飛行距離を視覚化しました。
  • 更新: 「Uniques Tools」ダイアログに新機能の説明を追加しました。

バージョン 1.5.1

  • 新規: Uniques Toolsダイアログに「最後の位置を検索」ボタンを追加し、最後にドローンで訪れたポータルにズームできるようにしました。
  • 修正: 画面外のポータルに対してポータル詳細APIを使用することで、ポータルへのズームロジックをより信頼性の高いものに修正しました。
