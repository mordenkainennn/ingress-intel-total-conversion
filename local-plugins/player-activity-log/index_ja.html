Player Activity Log

このプラグインは公開 COMM を監視し、プレイヤー行動ログをブラウザに保存します。

0.8.1 の更新

  • インポート前にプレビュー統計を表示し、確認後に保存するようになりました。
  • JSON インポート/エクスポートをサポート。
  • CSV はエクスポートのみ対応（インポートは JSON のみ）。
  • 重複除去は GUID 優先、GUID がない場合は指紋（player/time/activity/lat/lng）で判定。

使い方

  1. ツールボックスの Activity Log を開く。
  2. Import で .json を選択。
  3. 表示されるプレビュー（追加件数・重複件数など）を確認。
  4. 確認後にマージして保存。
  5. 共有は Export JSON、表計算用途は Export CSV を使用。
データはローカル保存です。端末間同期はありません。定期的にエクスポートしてバックアップしてください。
