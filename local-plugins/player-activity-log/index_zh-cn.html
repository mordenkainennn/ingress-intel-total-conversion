玩家活动日志 (Player Activity Log)

该插件会监听公开 COMM，并将解析后的玩家活动记录在浏览器本地。

0.8.1 更新

  • 导入前会显示预览统计，并在确认后才写入本地。
  • 支持 JSON 导入/导出。
  • CSV 仅支持导出（导入仅支持 JSON）。
  • 去重策略：优先 GUID；若无 GUID，使用指纹（player/time/activity/lat/lng）。

使用方法

  1. 在工具箱打开 Activity Log。
  2. 点击 Import 并选择 .json 文件。
  3. 先查看预览信息（新增数、重复数等）。
  4. 确认后执行合并并写入本地。
  5. 交换数据推荐 Export JSON；表格分析使用 Export CSV。
数据仅保存在本地，不会跨设备自动同步。请定期导出备份。
