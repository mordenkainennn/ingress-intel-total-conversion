Player Activity Log

This plugin monitors public COMM messages and stores parsed player activities in your browser.

Use it to review actions over time, visualize movement paths, and exchange merged data between users.

What's New in 0.8.1

  • Added JSON export and JSON import.
  • Kept CSV export (import is JSON-only).
  • Import now supports merge with de-duplication: GUID first, then fallback fingerprint (player + time + activity + lat + lng).
  • Import now shows a preview summary and asks for confirmation before writing data.

Features

  • Automatic Logging from COMM

    Passively listens to public COMM events and records supported actions (capture, deploy, link, field, etc.). MACHINA is ignored.

  • No 3-Day Cutoff

    The plugin now records all activity messages returned by Intel COMM (subject to what Intel returns in each fetch).

  • Interactive Activity Viewer

    Open from toolbox and browse players on the left with detailed activity entries on the right. Search supports player ID/name filtering.

  • Trail Rendering

    Select up to 3 players and draw map trails. Lines connect all loaded portals in chronological order and use dashed style. Markers stay fully visible.

  • Import and Export

    Available buttons in the dialog header:

    • Import: Import .json, review preview stats, then confirm to merge.
    • Export JSON: Recommended exchange format for complete structured data.
    • Export CSV: Spreadsheet-friendly export.

  • Local Storage

    All data is stored locally in browser localStorage.

How to Use

  1. Open Activity Log from IITC toolbox.
  2. Use search box to filter players by ID/name.
  3. Select players and click Draw Trails / Clear Trails.
  4. Use Import to load JSON, review preview counts, and confirm merge.
  5. Use Export JSON for sharing, or Export CSV for spreadsheet analysis.
  6. Use Clear All or per-player delete (×) if needed.

Important: Data is local only and not synced across devices/accounts. Back up regularly using export features.

Compatibility Note: Trail drawing overlaps with the official Player activity tracker layer. Disable one of them if needed.
